这时候再创建MyArray的衍生对象就将 Symbol.species返回的函数当做构造函数(constructor)
什么是衍生对象？

这里生成的对象b就是衍生对象. Symbol.species的作用在于，实例对象在运行过程中，需要再次调用自身的构造函数时，会调用Symbol.species返回的构造函数
注意new一个MyArray对象的时候不会调用Symbol.species指定的构造函数
